# Relatórios de Implementação por IA

Este diretório é preenchido pelo agente executor.

O relatório deve registrar:

- task ID;
- branch;
- commit-base;
- commits produzidos;
- arquivos alterados;
- resumo técnico;
- testes executados;
- resultados;
- limitações;
- riscos;
- comandos sensíveis não executados;
- confirmação de ausência de merge automático.

O executor não deve editar a especificação nem a revisão.
