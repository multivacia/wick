# Auditoria R2

Tente encontrar:
- fórmula ambígua;
- divisão por zero;
- candle aberto;
- look-ahead;
- confirmação processada cedo;
- tendência embutida e não registrada;
- exclusividade indevida;
- versão ausente;
- idempotência quebrada;
- falso positivo em limites;
- detector sem negativo;
- padrão multi-candle com âncora errada;
- contexto calculado com candle futuro.
