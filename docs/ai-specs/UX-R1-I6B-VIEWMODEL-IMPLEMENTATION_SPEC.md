## Architecture

- Framework-agnostic TypeScript modules (no React/router/fetch).
- Deterministic pure builders; optional `ViewModelClock.nowIso` for freshness.
- Missing metrics/timestamps remain `null` with explicit availability.
- Semantic inequalities: NOT_READY / BLOCKED / DEFERRED ≠ FAULT; red only for fault.
