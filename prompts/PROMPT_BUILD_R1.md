# Prompt — Implementar R1

Leia `CLAUDE.md`, `docs/PROJECT.md`, `docs/architecture/DATA_MODEL.md`, `docs/architecture/DATA_QUALITY.md` e `docs/releases/R1_SPEC.md`.

Crie uma branch `feature/r1-ingestion`.

Faça uma checagem de realidade no repositório. Se estiver vazio, inicialize. Não avance para R2.

Implemente integralmente a R1. Trabalhe autonomamente até que os critérios de aceite estejam satisfeitos. Corrija testes e lint sem pedir autorização.

Pare e solicite interação humana somente nas condições descritas em `CLAUDE.md` e na especificação.

Ao final:
1. execute todos os comandos mínimos;
2. gere relatório de qualidade de uma execução de exemplo;
3. atualize README;
4. faça commits organizados;
5. apresente o relatório final padrão;
6. não faça merge em outra branch.
